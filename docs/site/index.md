# Posipaka

**Персональний AI-агент для месенджерів.**

Posipaka живе на вашому сервері та спілкується через Telegram, Discord, Slack, WhatsApp або Signal. Він не просто відповідає — він **діє**: читає пошту, керує календарем, шукає інформацію, виконує код.

## Швидкий старт

```bash
curl -fsSL https://posipaka.dev/install.sh | bash
```

Або через pip:

```bash
pip install posipaka
posipaka setup
posipaka start
```

## Ключові можливості

- **5 месенджерів**: Telegram, Discord, Slack, WhatsApp, Signal
- **30+ інтеграцій**: Gmail, Calendar, GitHub, Shell, Browser, Weather, News...
- **30+ персон**: від тренера до DevOps інженера
- **4 шари пам'яті**: RAM → SQLite → MEMORY.md → ChromaDB
- **Безпека**: injection detection, audit log, approval gates, shell sandbox
- **Мультимовність**: 20+ мов з автовизначенням
